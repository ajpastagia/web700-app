I will be demonstrating HTML5 principles and techniques, as well as routing in Express.

Introduction to This Website

This site consists of 3 static views (Home, About, HTML Demo) as well as 5 routes that supply data. These include:

  • All Students
  • Students by Course
  • A Single Student by Student Number
  • All Managers
  • All Courses

Explore the site using the links above to interact with the data or navigate to the other pages.

Introduction to HTML5

This site utilizes HTML5, a markup language used for structuring and presenting content on the World Wide Web. HTML5 includes many new features like multimedia support, semantic elements, and improved APIs for better web functionality.
